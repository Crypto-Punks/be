#### Running the app

Lifecycle scripts included in crypto-punk-be:

 * test -
    npm run jest

 * start -
    node server.js

available via `npm run-script`:

 * lint -
    eslint .

 * retest -
    npm run lint

 * jest -
    jest --runInBand

 * test:coverage -
    npm run test -- --coverage

 * test:watch -
    npm run jest -- --watchAll

 * test:verbose -
    npm run test -- --verbose

 * start:watch
    nodemon server.js
